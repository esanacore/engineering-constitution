# Example Source (Demo Fixture)

This is a placeholder file, not a real book or article. It exists so
`scripts/check_source_summaries.sh scan` has something to report as `OK`
right out of the box, and so `sources/summaries/example/sample-source.md`
shows a fully worked summary you can copy the shape of.

If this were a real source, its actual text or excerpt would go here instead.

Delete the whole `example/` folder — this one under `sources/raw/` and the
matching one under `sources/summaries/` — once you've dropped in your first
real source and no longer need the demo. Leaving it in place is harmless
either way; `scripts/check_source_summaries.sh scan` will just keep reporting
it as `OK`.
