# Changelog Release Manager

Use this skill when the `Unreleased` section of `CHANGELOG.md` has accumulated user-facing changes. Bump the `VERSION` file, tag the release, and move changes to a versioned header.

**Constitution Alignment**: This skill strictly enforces the principles laid out in Eric's Engineering Constitution. Always adhere to the established workflows when applying this skill.
