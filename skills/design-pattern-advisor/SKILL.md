# Design Pattern Advisor

Use this skill to apply patterns like Factory, Decorator, or Adapter. Only apply them when a concrete problem exists (e.g., swapping families of objects), not just to name a pattern.

**Constitution Alignment**: This skill strictly enforces the principles laid out in Eric's Engineering Constitution. Always adhere to the established workflows when applying this skill.
