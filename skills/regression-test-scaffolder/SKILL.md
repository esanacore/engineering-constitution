# Regression Test Scaffolder

Use this skill whenever fixing a bug. Before or immediately after fixing the issue, write a test that explicitly reproduces the failure state to prevent regressions.

**Constitution Alignment**: This skill strictly enforces the principles laid out in Eric's Engineering Constitution. Always adhere to the established workflows when applying this skill.
