# Requirements Traceability Mapper

Use this skill when adding or modifying product requirements. Map the requirement ID to the specific tests that verify it in `docs/REQUIREMENTS_TRACEABILITY.md`. Ensure every ID maps to a verifying test.

**Constitution Alignment**: This skill strictly enforces the principles laid out in Eric's Engineering Constitution. Always adhere to the established workflows when applying this skill.
