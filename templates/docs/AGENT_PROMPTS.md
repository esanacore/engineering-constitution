# Agent Prompts

This document provides copyable prompts to help humans interact effectively with AI agents in this repository.

## Task Planning

```text
Review the current TODO.md and ARCHITECTURE.md. I want to implement [Feature X]. Create a step-by-step implementation plan that follows our Engineering Constitution.
```

## Bug Investigation

```text
The system is failing with [Error]. Review the relevant code and docs/TROUBLESHOOTING.md to identify the root cause and propose a fix.
```

## Refactoring

```text
Review [File/Module] for technical debt. Suggest refactoring opportunities and record them in TODO.md.
```

## Code Review

```text
Review my recent changes in [PR/Branch]. Ensure they adhere to Principle 2 (Testing) and Principle 5 (Security) of our Constitution.
```
