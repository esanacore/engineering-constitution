# Project Engineering Rules

This repository follows [Eric's Engineering Constitution](constitution/CONSTITUTION.md).

## Instructions for AI Agents

As an agent in this workspace, you MUST:
1. **Context Initialization**: Read `constitution/CONSTITUTION.md` and `constitution/AI_WORKFLOW.md`.
2. **Project Rules**: Adhere to the conventions in `AGENTS.md`.
3. **Continuous Updates**: Maintain `TODO.md` and `CHANGELOG.md` for every task.
4. **Architecture**: Follow the patterns in `docs/adr/` and `docs/ARCHITECTURE.md`.
5. **Testing**: Never consider a task complete without running or adding automated tests.

If you are using a tool that does not auto-load instructions, please copy this entire file into your system prompt.
